Cannibal Corpse Tour Project
Project title
Cannibal Corpse Tour Fan Page

Brief description of the project
I made this website to show the tour dates and information about the band Cannibal Corpse. It has a navigation menu, a table with concert schedules, a section about the band, and a gallery with their albums. The design is dark and responsive so it looks good on phones and computers.

Technologies used
HTML5

CSS3

Project structure
index.html: The main file with the content.

ms.css: The file with all the styles and design.

How to run or view the project locally
Save the index.html and ms.css files in the same folder.

Open the index.html file in your web browser.
